From Vacuum Tubes to Transistors

حاسوب ENIAC (1945)

THE DRIVING FORCE

Moore's Law

القانون الهيكلي السحري الذي تسبب في تصغير الأجهزة وزيادة سرعتها بشكل مرعب:

قانون مور • Moore's Law
ينص على أن عدد الترانزستورات على الشريحة الإلكترونية يتضاعف تقريباً كل سنتين، مما يعني تضاعف السرعة وصغر الحجم الفائق!

IBM 7090 (1959)
نتيجة التطور المباشر؛ أول حاسوب تجاري ضخم يعتمد بالكامل على الترانزستور بدلاً من الأنابيب.

MODERN ERA

Modern Device Architectures

بفضل ملايين الترانزستورات داخل رقاقة واحدة، تنوعت أشكال الحواسيب الحديثة في حياتنا:

1. الحاسوب المكتبي (Desktop PC)
قوة معالجة خارقة، مساحة تبريد واسعة، وأداء مستقر للمهام الثقيلة والألعاب.

2. الحاسوب المحمول (Laptop)
هندسة ضغط متطورة؛ لوحة أم وبطارية ومعالج في مساحة مسطحة خفيفة الوزن للتنقل.

3. الهاتف الذكي (Smartphone)
حاسوب متكامل فائق الصغر في جيبك! معالج SoC يدمج الرسوميات والذكاء الاصطناعي والاتصالات بدقة متناهية.

4. الساعة الذكية (Smartwatch)
قمة التصغير الهندسي؛ معالجات وحساسات حيوية دقيقة تعمل بطاقة منخفضة جداً على معصم يدك.

LOGIC UNITS

Bit & Byte Architecture

كيف تترجم الداتا داخل الترانزستور؟

وجود كهرباء = ON / 1
غياب كهرباء = OFF / 0

Bit = خانة فردية واحدة (0 أو 1).
Byte = تجميعة من 8 Bits متجاورة تُشكل كلمة برمجية مخزنة.

ABSTRACTION

Translators

اللغات العليا قريبة من الإنجليزية البشرية، ونحتاج وسيطاً للترجمة إلى لغة الآلة:

Python / C++ → [Translator] → Machine Code (0101)
Interpreter (المفسر الفوري)

يترجم سطر بسطر فوراً أثناء التشغيل.
مثال: لغة Python (سهلة ومباشرة).

Compiler (المترجم الكلي)

يترجم الملف كاملاً دفعة واحدة قبل التشغيل لإنتاج ملف سريع.
مثال: لغة C++ (فائقة السرعة).
